# Portfolio
A personal portfolio built with React + Vite, designed to showcase projects, experience, and design philosophy.
The site focuses on a clean, minimalistic UI with soft shadows, rounded corners, and a responsive layout that adapts smoothly across all screen sizes.

## Features
- Fully responsive design across all screen sizes
- Modern UI with a minimal, soft aesthetic 
- Smooth navigation with a custom Navbar
- Clear page structure for About, Projects, and single project pages.
- Animated tech marquee slider

## Tech Stack
- React + Vite
- CSS 
- React Router
- Netlify (Deployment)

## Notes
This portfolio is actively updated as I add new projects and case studies.
